DREAMPARTNER

Emotion‑based compatibility website with an integrated game that turns AI‑driven partner analysis into an immersive, playful experience.

DESCRIPTION

DreamPartner is a web‑based platform that combines an interactive game built with Godot Engine and machine learning models trained on emotional datasets to analyze and predict partner compatibility.

It uses a full‑stack architecture with Python on the backend, React.js on the frontend, and MySQL for data management, delivering a responsive, engaging, AI‑powered experience around relationships and emotions.

TECH STACK

  • React.js frontend
  • Python backend
  • TensorFlow & Scikit‑learn
  • Godot Engine (game)
  • MySQL database

HIGHLIGHTS

  • Emotion‑driven compatibility scoring using ML models.
  • Interactive Godot‑based game integrated into the website.
  • End‑to‑end full‑stack flow: Python backend, React.js UI, MySQL data.
  • Responsive, immersive interface tuned for user engagement.
